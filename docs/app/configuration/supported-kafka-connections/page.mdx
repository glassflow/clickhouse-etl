# Supported Kafka Connections

GlassFlow supports connecting to Kafka clusters with multiple protocols and authentication methods. This page lists the supported options and how to configure them.

<Callout type="info" emoji="ℹ️">
Configuration examples below refer to the pipeline source connection parameters. You can configure these via the Web UI or API.
</Callout>

## Connection Protocols

Set the **Security Protocol** ([`protocol`](/configuration/pipeline-json-reference#kafka-connection-parameters)) according to your broker configuration:

- **PLAINTEXT**
- **SASL_PLAINTEXT**
- **SSL**
- **SASL_SSL**

## SASL Authentication

Set one of the following **Authentication Methods** ([`mechanism`](/configuration/pipeline-json-reference#kafka-connection-parameters)):

- **PLAIN**
- **SCRAM-SHA-256**
- **SCRAM-SHA-512**

On the UI, you can disable authentication entirely by selecting "No authentication" in the authentication method dropdown. On the Python SDK, you can set the `skip_auth` flag to `true`.

When authentication is enabled, the following fields are required:
  - `mechanism` (one of the above)
  - `username`
  - `password`

## TLS Configuration

Enable TLS via protocol SSL/SASL_SSL and optionally supply root_ca to trust a custom CA.

Client certificate authentication (mTLS) is not supported.

## Example Configurations

### PLAINTEXT (no auth)

<Tabs items={['Web UI', 'Python SDK']} groupId="kafka-conn" storageKey="kafka-conn-tab">
  <Tabs.Tab>
    <img alt="PLAINTEXT (no auth) example" src="/assets/ui_kafka_connection_plaintext_no_auth.png" />
  </Tabs.Tab>
  <Tabs.Tab>
    ```json
    "connection_params": {
      "brokers": ["kafka-1:9092"],
      "protocol": "PLAINTEXT",
      "skip_auth": true
    }
    ```
  </Tabs.Tab>
</Tabs>

### SASL/PLAIN over TLS

<Tabs items={['Web UI', 'Python SDK']} groupId="kafka-conn" storageKey="kafka-conn-tab">
  <Tabs.Tab>
    <img alt="SASL/PLAIN over TLS example" src="/assets/ui_kafka_connection_sasl_plain_over_tls.png" />
  </Tabs.Tab>
  <Tabs.Tab>
    ```json
    "connection_params": {
      "brokers": ["kafka-1:9093"],
      "protocol": "SASL_SSL",
      "skip_auth": false,
      "mechanism": "PLAIN",
      "username": "user",
      "password": "pass",
      "root_ca": "-----BEGIN CERTIFICATE-----\n...\n-----END CERTIFICATE-----\n"
    }
    ```
  </Tabs.Tab>
</Tabs>
