# DynamoDB

<Tier badge="enterprise" />

DynamoDB is a fully managed NoSQL key-value and document database by AWS. It provides low-latency performance at any scale with automatic scaling and high availability.

## Using DynamoDB with GlassFlow

GlassFlow can ingest changes from DynamoDB into ClickHouse with the same pipeline capabilities available to all GlassFlow sources, including deduplication, joins, filtering, and stateless transformations.
