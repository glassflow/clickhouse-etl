# Segment

Segment is a customer data platform that collects, unifies, and routes user event data to downstream systems. It simplifies analytics, marketing, and data infrastructure pipelines.

## Using Segment with GlassFlow

GlassFlow can ingest data from Segment into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
