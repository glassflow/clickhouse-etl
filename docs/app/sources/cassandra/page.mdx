# Apache Cassandra®

<Tier badge="enterprise" />

Cassandra is a distributed NoSQL database designed for high availability and scalability across multiple nodes. It is optimized for large-scale, write-heavy workloads with no single point of failure.

## Using Apache Cassandra with GlassFlow

GlassFlow can ingest changes from Apache Cassandra into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
