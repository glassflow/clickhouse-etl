# AWS Kinesis

<Tier badge="enterprise" />

Kinesis is a real-time data streaming service that collects, processes, and analyzes large volumes of data. It is commonly used for log ingestion, metrics, and event pipelines at scale.

## Using AWS Kinesis with GlassFlow

GlassFlow can ingest data from AWS Kinesis Data Streams into ClickHouse with the same pipeline capabilities available to all GlassFlow sources, including deduplication, joins, filtering, and stateless transformations.
