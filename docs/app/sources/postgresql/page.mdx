# PostgreSQL

<Tier badge="enterprise" />

PostgreSQL is an advanced open-source relational database known for reliability and extensibility. It supports complex queries, ACID transactions, and modern features like JSON and vector extensions.

## Using PostgreSQL with GlassFlow

GlassFlow can ingest changes from PostgreSQL into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
