# Azure Event Hubs

<Tier badge="enterprise" />

Event Hubs is a large-scale event ingestion service by Azure designed for streaming millions of events per second. It is commonly used for telemetry pipelines, logging, and real-time analytics.

## Using Azure Event Hubs with GlassFlow

GlassFlow can ingest data from Azure Event Hubs into ClickHouse with the same pipeline capabilities available to all GlassFlow sources, including deduplication, joins, filtering, and stateless transformations.
