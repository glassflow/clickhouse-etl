# MariaDB

<Tier badge="enterprise" />

MariaDB is an open-source relational database forked from MySQL, designed for performance and scalability. It offers enhanced storage engines and compatibility with MySQL ecosystems.

## Using MariaDB with GlassFlow

GlassFlow can ingest changes from MariaDB into ClickHouse with the same pipeline capabilities available to all GlassFlow sources, including deduplication, joins, filtering, and stateless transformations.
