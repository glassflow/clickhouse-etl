# Amazon S3

<Tier badge="enterprise" />

Amazon S3 is a highly durable and scalable object storage service used as the backbone for data lakes. It stores structured and unstructured data and integrates with analytics and processing systems.

## Using Amazon S3 with GlassFlow

GlassFlow can ingest data from Amazon S3 into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, filtering, and stateless transformations.
