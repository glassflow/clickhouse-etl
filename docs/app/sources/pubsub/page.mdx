# Google Pub/Sub

<Tier badge="enterprise" />

Google Pub/Sub is a fully managed messaging service by Google Cloud for real-time event ingestion and delivery. It enables scalable, asynchronous communication between services using publish–subscribe patterns.

## Using Google Pub/Sub with GlassFlow

GlassFlow can ingest data from Google Cloud Pub/Sub topics into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
