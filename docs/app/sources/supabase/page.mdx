# Supabase

Supabase is an open-source backend platform built on PostgreSQL, offering APIs, authentication, and real-time capabilities. It provides a Firebase-like experience with SQL-based data storage.

## Using Supabase with GlassFlow

GlassFlow can ingest data from Supabase into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
