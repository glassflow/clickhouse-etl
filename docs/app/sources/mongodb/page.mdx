# MongoDB

MongoDB is a NoSQL document-oriented database that stores data in flexible JSON-like documents. It enables rapid development and horizontal scaling for modern applications.

## Using MongoDB with GlassFlow

GlassFlow can ingest changes from MongoDB into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
