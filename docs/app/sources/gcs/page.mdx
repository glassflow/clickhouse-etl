# Google Cloud Storage

GCS is a scalable object storage service for storing and retrieving large amounts of data. It is commonly used as a data lake backend for analytics and machine learning.

## Using GCS with GlassFlow

GlassFlow can ingest data from Google Cloud Storage into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, filtering, and stateless transformations.
