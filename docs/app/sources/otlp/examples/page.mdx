# OTLP Source Examples

## Quick Start

<Steps>

### Create the ClickHouse table

Run the following in your ClickHouse client to create a logs table (see [ClickHouse Tables](/sources/otlp/clickhouse-tables) for traces and metrics schemas):

```sql
CREATE TABLE IF NOT EXISTS otel_logs
(
    `Timestamp` DateTime64(9) CODEC(Delta(8), ZSTD(1)),
    `TimestampTime` DateTime DEFAULT toDateTime(Timestamp),
    `TraceId` String CODEC(ZSTD(1)),
    `SpanId` String CODEC(ZSTD(1)),
    `TraceFlags` UInt8,
    `SeverityText` LowCardinality(String) CODEC(ZSTD(1)),
    `SeverityNumber` UInt8,
    `ServiceName` LowCardinality(String) CODEC(ZSTD(1)),
    `Body` String CODEC(ZSTD(1)),
    `ResourceAttributes` Map(LowCardinality(String), String) CODEC(ZSTD(1)),
    `ScopeName` String CODEC(ZSTD(1)),
    `ScopeVersion` LowCardinality(String) CODEC(ZSTD(1)),
    `ScopeAttributes` Map(LowCardinality(String), String) CODEC(ZSTD(1)),
    `LogAttributes` Map(LowCardinality(String), String) CODEC(ZSTD(1)),
    INDEX idx_trace_id TraceId TYPE bloom_filter(0.001) GRANULARITY 1,
    INDEX idx_res_attr_key mapKeys(ResourceAttributes) TYPE bloom_filter(0.01) GRANULARITY 1,
    INDEX idx_res_attr_value mapValues(ResourceAttributes) TYPE bloom_filter(0.01) GRANULARITY 1,
    INDEX idx_scope_attr_key mapKeys(ScopeAttributes) TYPE bloom_filter(0.01) GRANULARITY 1,
    INDEX idx_scope_attr_value mapValues(ScopeAttributes) TYPE bloom_filter(0.01) GRANULARITY 1,
    INDEX idx_log_attr_key mapKeys(LogAttributes) TYPE bloom_filter(0.01) GRANULARITY 1,
    INDEX idx_log_attr_value mapValues(LogAttributes) TYPE bloom_filter(0.01) GRANULARITY 1,
    INDEX idx_body Body TYPE tokenbf_v1(32768, 3, 0) GRANULARITY 8
)
ENGINE = MergeTree()
PARTITION BY toDate(TimestampTime)
PRIMARY KEY (ServiceName, TimestampTime)
ORDER BY (ServiceName, TimestampTime, Timestamp);
```

### Define the pipeline configuration

Create a pipeline configuration file with your OTLP source, ClickHouse sink, and schema mapping:

```json
{
  "version": "v3",
  "pipeline_id": "otel-logs-pipeline",
  "source": {
    "type": "otlp.logs",
    "deduplication": {
      "enabled": true,
      "key": "trace_id",
      "time_window": "5m"
    }
  },
  "join": {"enabled": false},
  "sink": {
    "type": "clickhouse",
    "host": "clickhouse.example.com",
    "port": "9000",
    "database": "otel",
    "username": "default",
    "password": "mysecret",
    "secure": true,
    "table": "otel_logs"
  },
  "schema": {
    "fields": [
      {"source_id": "otlp.logs", "name": "timestamp",            "type": "string", "column_name": "Timestamp",          "column_type": "DateTime64(9)"},
      {"source_id": "otlp.logs", "name": "trace_id",             "type": "string", "column_name": "TraceId",             "column_type": "String"},
      {"source_id": "otlp.logs", "name": "span_id",              "type": "string", "column_name": "SpanId",              "column_type": "String"},
      {"source_id": "otlp.logs", "name": "flags",                "type": "uint",   "column_name": "TraceFlags",          "column_type": "UInt8"},
      {"source_id": "otlp.logs", "name": "severity_text",        "type": "string", "column_name": "SeverityText",        "column_type": "LowCardinality(String)"},
      {"source_id": "otlp.logs", "name": "severity_number",      "type": "uint",   "column_name": "SeverityNumber",      "column_type": "UInt8"},
      {"source_id": "otlp.logs", "name": "body",                 "type": "string", "column_name": "Body",                "column_type": "String"},
      {"source_id": "otlp.logs", "name": "resource_attributes.service.name",  "type": "string", "column_name": "ServiceName",         "column_type": "LowCardinality(String)"},
      {"source_id": "otlp.logs", "name": "resource_attributes",  "type": "map",    "column_name": "ResourceAttributes",  "column_type": "Map(LowCardinality(String), String)"},
      {"source_id": "otlp.logs", "name": "scope_name",           "type": "string", "column_name": "ScopeName",           "column_type": "String"},
      {"source_id": "otlp.logs", "name": "scope_version",        "type": "string", "column_name": "ScopeVersion",        "column_type": "LowCardinality(String)"},
      {"source_id": "otlp.logs", "name": "scope_attributes",     "type": "map",    "column_name": "ScopeAttributes",     "column_type": "Map(LowCardinality(String), String)"},
      {"source_id": "otlp.logs", "name": "attributes",           "type": "map",    "column_name": "LogAttributes",       "column_type": "Map(LowCardinality(String), String)"}
    ]
  }
}
```

### Create the pipeline

Using the [Python SDK](/usage-guide/python-sdk):

```python
from glassflow.etl import Client

client = Client(host="http://localhost:30180")
pipeline = client.create_pipeline(pipeline_config_json_path="pipeline_config.json")
```

### Send data to the pipeline

Point an OpenTelemetry Collector or SDK exporter at the GlassFlow OTLP receiver. See [Sending Data](/sources/otlp/sending-data) for detailed configuration options. A minimal OTel Collector config:

```yaml
exporters:
  otlphttp:
    endpoint: http://<glassflow-otlp-receiver>:4318
    headers:
      x-glassflow-pipeline-id: "otel-logs-pipeline"

service:
  pipelines:
    logs:
      receivers: [otlp]
      processors: [batch]
      exporters: [otlphttp]
```
