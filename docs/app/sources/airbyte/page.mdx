# Airbyte

Airbyte is an open-source data integration platform that syncs data from various sources into warehouses and lakes. It supports ELT pipelines with a large library of connectors.

## Using Airbyte with GlassFlow

GlassFlow can ingest data from Airbyte connectors into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
