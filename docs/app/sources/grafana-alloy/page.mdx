# Grafana Alloy

Grafana Alloy is a telemetry collector built on OpenTelemetry that gathers and forwards logs, metrics, and traces. It integrates tightly with the Grafana ecosystem for observability pipelines.

## Using Grafana Alloy with GlassFlow

GlassFlow can ingest telemetry forwarded by Grafana Alloy into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, filtering, and stateless transformations.
