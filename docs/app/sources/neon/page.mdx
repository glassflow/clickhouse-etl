# Neon

Neon is a serverless PostgreSQL platform that separates storage and compute for efficient scaling. It enables instant branching, autoscaling, and cost-efficient database operations.

## Using Neon with GlassFlow

GlassFlow can ingest changes from Neon into ClickHouse with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
