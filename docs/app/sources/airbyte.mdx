# Airbyte

Airbyte is an open-source data integration platform that syncs data from various sources into warehouses and lakes. It supports ELT pipelines with a large library of connectors.

## Using Airbyte with GlassFlow

GlassFlow can integrate with Airbyte to bring its extensive connector ecosystem into a real-time path to ClickHouse, with the same pipeline capabilities available to all GlassFlow sources — deduplication, joins, filtering, and stateless transformations.
