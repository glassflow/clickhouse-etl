# GlassFlow on AWS EC2

This guide will walk you through the process of installing GlassFlow on an AWS EC2 instance.

## Prerequisites

- AWS account with enought permissions to create EC2 instances and security groups.

## Installation

1. Create a new EC2 instance with Linux 2 AMI.

2. Update instance security group to allow inbound traffic on port `8080`.

3. Log in to the instance and install git and docker:
```bash
sudo yum update -y
sudo yum install -y amazon-linux-extras git 
sudo amazon-linux-extras install docker -y
sudo systemctl start docker  
sudo usermod -a -G docker $USER  
sudo systemctl enable docker
```

4. Install docker-compose:
```bash
sudo curl -L https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m) -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose
```

5. Restart terminal and verify docker installation:
```bash
docker info
docker-compose version
```

6. Clone the GlassFlow repository:
```bash
git clone https://github.com/glassflow/clickhouse-etl.git
```

7. Start the services using Docker Compose:
```bash
cd clickhouse-etl
docker compose up -d
```

8. Access the web interface at `http://<instance-public-ip>:8080`.
